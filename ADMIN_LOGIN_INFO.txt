⚙️  API FUNCTIONS (convex/auth.ts)

login()              → Authenticate user
getCurrentAdmin()    → Verify session token
logout()             → End session & clear credentials
createSuperAdmin()   → Create first admin account
createAdmin()        → Create additional admin (super admin only)
changePassword()     → Update admin password
listAdmins()         → List all admins (super admin only)
deactivateAdmin()    → Disable account (super admin only)

🐛 TROUBLESHOOTING

"Invalid email or password"
  → Check credentials are exact (case-sensitive email)
  → Verify account is active (isActive = true)

Can't access /admin/create
  → Login first at /auth/login
  → Check localStorage has adminToken

Session keeps expiring
  → Sessions are 30 days (not a bug!)
  → Login again to get fresh token

💡 TIPS & TRICKS

1. Use localStorage to check if logged in:
   const token = localStorage.getItem('adminToken')
   
2. Check admin role:
   const role = localStorage.getItem('adminRole')
   
3. View all sessions in browser:
   F12 → Application → Local Storage → http://localhost:3000
   
4. Check Convex logs:
   npx convex logs
   
5. Clear all admin data:
   localStorage.clear()  // Then login again
